# VDumpling Dynasty Website

This is the official website for VDumpling Dynasty, a chain of Nepalese restaurants serving authentic momos and other Himalayan delicacies in Bhubaneswar, India.

## Deployment Package

The website has been built and is ready for deployment. You can find the deployment package in:

- `vdd-website-deploy.zip` 

This ZIP file contains all the necessary files for hosting the website on platforms like Hostinger.

For detailed deployment instructions, please refer to:

- `DEPLOYMENT.md`

## Development

If you need to make changes to the website:

1. Clone this repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Build the production version: `npm run build`

## Technology Stack

- React.js
- Tailwind CSS
- Vite
